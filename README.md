# bc-snippet-code README

## 业务常用片段

### vue文件

- `vue3`  快速生成一个页面默认结构
- `admin.list` 生成一个标准后台页面，上面搜索下面列表
- `admin.modal` 生成弹窗模板

### ts,tsx

- `admin.searchConfig` 配合`admin.list`使用，快速生成列表和搜索项配置
- `admin.modalConfig` 配合 `admin.modal`使用，用于函数式调用模板

### ts
- `admin.router` 生成一个基本的路由结构，只对ts文件起作用


**Enjoy!**
